## What Is This?

This repository contains a complete methodology for building persistent, verifiable knowledge systems that work across the stateless gap between human memory and AI context windows.

It emerged not from academic research, but from practical necessity—an electrician working 6am to 6pm on industrial projects, developing systems to capture and compress knowledge that would otherwise be lost.

## Core Principles

### Temet Nosce — Know Thyself
Self-knowledge is operational necessity. A system that doesn't know what it is cannot know what it should do.

### 100% Truth
The methodology only works if every component is honest. Hallucination breaks verification. Exaggeration corrupts compression.

### Constraint-Based Amplification
Constraints are not limitations—they are structures that focus power. A river without banks is a swamp.

### Three-Witness Verification
No single source establishes truth. Verification requires convergence across three independent witnesses.

### 25:1 Compression Ratio
Raw data crystallizes into structure at approximately 25:1. Every 25 units of input produces 1 unit of verified, structured knowledge.

---

## The Six-Step Methodology

1. **Define Scope** — Establish boundaries before gathering
2. **Gather with Intent** — Every piece connects to the defined scope
3. **Compress to Structure** — Apply the 25:1 ratio
4. **Verify Through Witnesses** — Confirm with three independent sources
5. **Store for Retrieval** — Knowledge that can't be retrieved isn't knowledge
6. **Iterate** — Knowledge evolves; structures need revision
